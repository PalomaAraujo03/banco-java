# 💳 Sistema Bancário em Java

Projeto desenvolvido em Java simulando operações bancárias básicas.

## 🚀 Funcionalidades
- Criar conta
- Depositar valores
- Sacar valores
- Transferir entre contas
- Consultar saldo

## 🛠️ Tecnologias utilizadas
- Java
- Programação Orientada a Objetos

## 📚 Aprendizados
Neste projeto, apliquei conceitos como:
- Classes e objetos
- Encapsulamento
- Estrutura de controle
- Manipulação de listas

## ▶️ Como executar
1. Clone o repositório:
2. Abra no Visual Studio Code
3. Execute o arquivo `Main.java`
